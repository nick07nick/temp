# Техническое Задание: Разработка PCBA для "BikeFit Active Marker"

**Тип проекта:** Коммерческая электроника / IoT / Wearable
**Статус:** Готов к разводке (Схемотехника и BOM определены)
**Цель:** Создать файлы для производства (Gerber + Pick&Place) для заказа сборки на JLCPCB.

## 1. Параметры Платы (Mechanical Specs)

* **Форма:** Круглая (Round).
* **Диаметр:** 17.0 мм (Строго).
* **Толщина текстолита:** 0.8 мм или 1.0 мм (Для компактности).
* **Количество слоев:** 4 слоя (Signal / GND / PWR / Signal).
    * *Критично:* 4 слоя обязательны для контроля импеданса антенны 50 Ом в таком малом размере.
* **Монтаж:** Двусторонний (Double-sided).
    * TOP Side: MCU, Antenna, LED.
    * BOTTOM Side: Accel, Charger, Battery Contacts.

## 2. Схемотехника (Ключевые узлы)

### 2.1. MCU (Мозг)
* **Чип:** Nordic **nRF52832-CIAA** (WLCSP) или **nRF52832-QFAA** (QFN-48).
    * *Предпочтение:* QFN-48 (QFAA), так как он дешевле в сборке и надежнее, если влезет в 17мм. Если не влезает — WLCSP.
* **Clock:** Внешний кварц 32.768 kHz (обязателен для низкого потребления Bluetooth).

### 2.2. Питание (Power)
* **Вход:** 2 контакта (Pads) для Pogo-Pins зарядного кейса (VBUS 5V, GND).
* **Заряд:** Чип **MCP73831** (или аналог). Ток заряда: 30-40mA.
* **Стабилизация:** LDO 3.3V или 3.0V (с низким IQ < 2uA), например **XC6206**.
* **Батарея:** Площадки под пайку выводов Li-Ion LIR1654 / LIR2430.

### 2.3. Сенсоры и Периферия
* **Акселерометр:** ST **LIS2DH12TR** (LGA-12). Интерфейс I2C или SPI. Pin INT1 должен быть подключен к GPIO пробуждения MCU.
* **LED Driver (Dual Channel):**
    * Использовать сдвоенный N-MOSFET (например, MBR360 в SOT-363).
    * **Канал А:** Резистор 100 Ом (0805).
    * **Канал B:** Резистор 33 Ом (1206, High Power!).
    * Управление с двух разных GPIO.
* **Status LED:** Синий SMD 0603.

### 2.4. Антенна (RF)
* **Тип:** Chip Antenna (Керамическая), например **Johanson 2450AT18B100**.
* **Требование:** Обязательный расчет Matching Network (Pi-network) и соблюдение рекомендаций производителя по полигонам земли (Ground Clearance).

### 2.5. Программирование
* Вывести на край платы (или в удобное место) тестовые точки (Test Points) под игольчатый адаптер:
    * VCC
    * GND
    * SWDIO
    * SWCLK

## 3. Требования к исполнителю
1.  Использовать **KiCad** (предпочтительно) или Altium Designer.
2.  Подобрать компоненты из библиотеки **JLCPCB Parts Library** (Basic/Extended), чтобы минимизировать время заказа.
3.  Результат работы:
    * Схема (PDF + Source).
    * Gerber файлы (для производства).
    * BOM (список деталей с номерами LCSC Part #).
    * Pick & Place файл (координаты центров компонентов).

## 4. Примечания
* Устройство носимое, критична компоновка.
* Нижняя сторона платы должна иметь большие площадки для пайки "+" и "-" аккумулятора LIR1654.
